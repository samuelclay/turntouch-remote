Detailed Description

Compatibility mode option. 

   This can be used with @ref sd_ble_opt_set to enable and disable
   compatibility modes. Compatibility modes are disabled by default.
Note
Compatibility mode 1 enables interoperability with devices that do not support a value of 0 for the WinOffset parameter in the Link Layer CONNECT_REQ packet.
Return values
NRF_SUCCESSSet successfully.
NRF_ERROR_INVALID_STATEWhen connection creation is ongoing while mode 1 is set.
