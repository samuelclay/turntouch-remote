Detailed Description

Application Programming Interface for S110 SoftDevice.

For more information about the S110 SoftDevice, please see the S110 nRF51822 SoftDevice Specification or the Introduction to the S110 SoftDevice which is located in the Documentation folder of this installation. For more information about Bluetooth® low energy, please visit http://developer.bluetooth.org/.
